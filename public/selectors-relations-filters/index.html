Селектор элемента

    background-color: blue;
}

}

Селектор id

    height: 25px;
}

    height: 100px;
}

Селектор класса

    width: 25px;
}

.wide {
    width: 100px;
}

Селектор прямого потомка

.wide > div {
    transform: rotate(45deg);
}

Селектор всех потомков

.wide  div {
    transform: rotate(45deg);
}

Сестренский селектор

.wide + .wide  {
    transform: rotate(45deg);
}

Селектор псевдокласса

Селектор структурного псевдокласса

Селектор структурного псевдокласса типа и псевдоэлемента
